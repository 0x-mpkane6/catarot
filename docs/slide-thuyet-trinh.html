09

Bảo mật theo mặc định

  • Mật khẩu PBKDF2 200.000 vòng + salt, so sánh hằng-thời-gian.
  • JWT fail-fast: production secret yếu → app từ chối khởi động.
  • Không tin user_id client gửi — lấy từ JWT. Chống IDOR.
  • Google OAuth: verify id_token với audience = GOOGLE_CLIENT_ID.
  • Rate limit cho /auth/* và /ask* (HTTP 429).
  • Quên mật khẩu chống enumeration (response giống nhau).
  • Upload chặt: whitelist đuôi, 25MB, xoá file tạm trong finally.
  • Che API key trong log; loại console.* ở production.

Embedding mở, tìm vector cực nhanh trên CPU — không cần train.

faster-whisper

Gemini · Groq

Chất lượng tiếng Việt tốt, free tier, Groq cực nhanh khi backup.

SQLAlchemy 2.0

Một codebase chạy cả SQLite (dev) lẫn Postgres (prod).

React 19 + Vite 7

HMR nhanh, code-split, vendor chunk, drop console prod.

Tiêu chí xuyên suốt: miễn phí, chạy CPU, ít dependency — phù hợp ràng buộc đồ án sinh viên.
